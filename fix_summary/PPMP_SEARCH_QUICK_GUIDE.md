# PPMP Item Search - Quick User Guide

## 🔍 What is it?

A search bar that helps you quickly find specific items when creating or editing a PPMP with many items.

## 📋 When does it appear?

The search bar **automatically appears** when you have **5 or more items** in your PPMP.

## 🎯 How to use it?

### Step 1: Add Items
1. Open PPMP creation or edit an existing PPMP
2. Add at least 5 items using the "Add Item" button
3. The search bar will appear at the top of the items section

### Step 2: Search
1. Click on the search box
2. Type what you're looking for:
   - Item description (e.g., "office supplies", "paper")
   - Type (e.g., "goods", "service")
   - Budget amount (e.g., "1000", "5000")
   - Unit (e.g., "box", "pcs")
   - Any other field content

### Step 3: View Results
- **Matching items** are highlighted with a **purple border**
- **Non-matching items** are hidden
- The page **automatically scrolls** to the first match
- See how many items matched below the search box

### Step 4: Clear Search
- Click the **X button** in the search box, OR
- Delete all text to show all items again

## 💡 Tips

### Quick Searches:
- **Find by description**: Type keywords from the item description
- **Find by budget**: Type the budget amount (e.g., "10000")
- **Find by type**: Type "goods", "service", or "infrastructure"
- **Find by source**: Type "IGF", "RAF", etc.

### Multiple Matches:
- If multiple items match, they all show up
- The page scrolls to the first one
- Scroll down to see other matches

### No Results:
- If no items match, you'll see "No items found"
- Try a different search term or clear the search

## 🎨 What to Look For

### Item Count Badge:
- Shows total number of items (e.g., "5 items", "12 items")
- Located next to "Procurement Items" heading

### Search Box:
- Purple border when focused
- Magnifying glass icon on the left
- X button on the right (when you type something)

### Matching Items:
- **Purple ring border** around the item card
- Item stays visible while others are hidden

### Results Info:
- **Purple text**: "Found X matching items" (when items match)
- **Red text**: "No items found" (when nothing matches)

## ⚡ Keyboard Tips

- **Type to search**: Just start typing in the search box
- **Clear search**: Click X button or delete all text
- **Scroll**: Use mouse wheel or arrow keys to navigate results

## ❓ FAQ

**Q: Why don't I see the search bar?**
A: The search bar only appears when you have 5 or more items. Add more items to see it.

**Q: Can I search by multiple fields at once?**
A: The search looks at all fields, so if you type "goods 1000", it will find items that have either "goods" OR "1000" in any field.

**Q: What happens to hidden items?**
A: They're just hidden temporarily. Clear the search to see them again. They're not deleted.

**Q: Does the search work when editing a PPMP?**
A: Yes! The search works the same way whether you're creating a new PPMP or editing an existing one.

**Q: Can I search by date?**
A: Not directly, but you can search by month/year if you know the format (e.g., "2026-01").
