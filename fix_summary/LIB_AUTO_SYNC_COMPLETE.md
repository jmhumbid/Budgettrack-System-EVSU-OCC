## Usage Flow

1. **Click "Auto-Generate from Allocations"**
   - Modal opens with year selector

2. **Select Year and Click "Generate LIB"**
   - System fetches approved allocations
   - System fetches existing custom items
   - Displays combined list in table

3. **Review Generated Items**
   - Allocation items shown with green badge (read-only)
   - Custom items shown with blue badge (editable)

4. **Add Custom Items (Optional)**
   - Click "Add Custom Item"
   - Fill in UACS code, description, amounts
   - Item saved to database
   - Appears in list immediately

5. **Edit/Delete Custom Items (Optional)**
   - Click edit icon on custom items
   - Modify values and save
   - Or delete if no longer needed

6. **Save LIB**
   - Click "Save LIB" button
   - System creates complete LIB record
   - All items (allocation + custom) saved together
   - LIB marked as "Final" (approved status)

## Technical Details

### Database Schema
```sql
CREATE TABLE lib_custom_items (
    id INT AUTO_INCREMENT PRIMARY KEY,
    department_id INT NOT NULL,
    year INT NOT NULL,
    uacs_code VARCHAR(50) NOT NULL,
    general_desc TEXT NOT NULL,
    total_amount DECIMAL(15,2) DEFAULT 0.00,
    quarter_1 DECIMAL(15,2) DEFAULT 0.00,
    quarter_2 DECIMAL(15,2) DEFAULT 0.00,
    quarter_3 DECIMAL(15,2) DEFAULT 0.00,
    quarter_4 DECIMAL(15,2) DEFAULT 0.00,
    created_by INT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP NULL,
    deleted_at TIMESTAMP NULL,
    deleted_by INT NULL,
    FOREIGN KEY (department_id) REFERENCES departments(id),
    FOREIGN KEY (created_by) REFERENCES users(id),
    FOREIGN KEY (deleted_by) REFERENCES users(id),
    INDEX idx_dept_year (department_id, year),
    INDEX idx_deleted (deleted_at)
);
```

## Troubleshooting

### Issue: "No approved allocations found"
**Solution:** Ensure allocations exist and are marked as "approved" status for the selected year and department.

### Issue: Custom item not saving
**Solution:** Check that:
- Department ID is set in session
- User has permission to create LIB
- All required fields are filled
- Database connection is active

### Issue: Grand total not updating
**Solution:** Check browser console for JavaScript errors. Ensure `displayAutoGeneratedItems()` is being called after modifications.

### Issue: Modal not appearing
**Solution:** Check z-index conflicts. Auto-generate modal uses z-50, custom item modal uses z-60.
