# Budget Allocation System - Quick Reference Guide

## 🚀 What's New?

### 5 Major Enhancements:
1. **Fiscal Year Selector** - Switch between years easily
2. **Simplified Interface** - Removed "Number of Students" field
3. **Numerical Percentages** - Enter "25" instead of "25%"
4. **Smart Sync** - Percentages auto-fill across departments
5. **Additional Amount** - Add special budget items with descriptions

---

## 📝 Quick Start Guide

### Creating a New Allocation:

```
1. Select Fiscal Year → Choose year from dropdown (defaults to current)
2. Select Department/Office → Search and select
3. Enter Budget Details:
   - For Departments: Total Tuition Fee
   - For Offices: Budget Allocated
4. Enter Percentages → Enter numbers only (e.g., 25, not 25%)
   ⚠️ Must total 100%
   💡 Will auto-fill for other departments
5. Add Deductions → Click "Add Deduction" if needed
6. Add Fiduciary Items → For offices/fiduciary departments
7. Additional Amount (Optional):
   - Enter amount
   - Add description
8. Generate Summary → Review calculations
9. Save Allocation → Confirm and save
```

---

## 🎯 Key Features

### Fiscal Year Selector
**Location:** Top left of page  
**Options:** 2021, 2022, 2023, 2024, 2025, 2026, 2027  
**Default:** Current year (2026)  
**Tip:** Selection is saved automatically

### Percentage Entry
**Format:** Numerical only (25, not 25%)  
**Validation:** Must total 100%  
**Sync:** Saved and applied to all departments  
**Override:** Can change per department

### Additional Amount
**When to Use:**
- Special equipment funds
- Emergency allocations
- Supplemental budgets
- One-time expenses

**Fields:**
- Amount (required if using)
- Description (recommended)

## 🆘 Troubleshooting

### Problem: Percentages not totaling 100%
**Solution:** Adjust values until total = 100%

### Problem: Percentages not syncing
**Solution:** Clear browser cache, try again

### Problem: Fiscal year not changing
**Solution:** Refresh page, select again

### Problem: Additional amount not in total
**Solution:** Check if amount is properly formatted (₱0.00)

### Problem: Can't save allocation
**Solution:** 
1. Check all required fields filled
2. Verify percentages = 100%
3. Check console for errors
4. Try refreshing page
