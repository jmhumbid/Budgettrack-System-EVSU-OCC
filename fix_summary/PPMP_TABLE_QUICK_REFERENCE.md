# PPMP Table Quick Reference Card

## 🎯 Quick Actions

| Action | How To | Result |
|--------|--------|--------|
| **View Details** | Click any row | Expands to show full information |
| **Hide Details** | Click expanded row | Collapses back to summary |
| **Expand All** | Click "Expand All" button | Opens all items at once |
| **Collapse All** | Click "Collapse All" button | Closes all items |
| **Print** | Click "Print" button | Shows full table automatically |

---

## 📊 Table Columns

### Screen View (5 Columns)
1. **#** - Item number
2. **General Description & Objective** - Main description (click to expand)
3. **Budget** - Estimated budget (green)
4. **Allocated** - Allocated funds (blue)
5. **Remarks** - Deduction notes (yellow if present)

### Print View (14 Columns)
Automatically shows all columns when printing:
- # | Description | Type | Qty | Unit | Mode | Pre-Proc | Start | End Ads | Delivery | Source | Budget | Allocated | Remarks

---

## 🔍 What's in Expanded Details?

When you click a row, you'll see:

### Basic Info
- **Type**: Goods, Service, or Infrastructure
- **Quantity**: Number and unit (e.g., "10 box")
- **Pre-Proc**: Pre-procurement conference (Yes/No)
- **Source**: Source of funds (IGF, RAF, etc.)

### Procurement Details
- **Recommended Mode**: Procurement method
- **Start Procurement**: Start date
- **End Ads/Posting**: Posting end date
- **Expected Delivery**: Delivery date

## 💡 Tips & Tricks

### For Quick Scanning
✅ Keep rows collapsed  
✅ Focus on descriptions and budgets  
✅ Use remarks column to identify deductions  

### For Detailed Review
✅ Click "Expand All"  
✅ Scroll through all details  
✅ Click "Collapse All" when done  

### For Specific Items
✅ Find item in collapsed view  
✅ Click to expand just that item  
✅ Review details  
✅ Click to collapse  

### For Printing
✅ Just click "Print"  
✅ No need to expand items  
✅ Full table shows automatically

## 🖨️ Printing Tips

### Before Printing
- No need to expand rows
- Check fiscal year filter
- Review page count

### Print Settings
- Orientation: Landscape (automatic)
- Paper: A4 or Letter
- Margins: 0.5 inch (automatic)
- Color: Recommended for headers

### What Prints
- Full 14-column table
- All items (no pagination)
- Grand total row
- Official header
- Generation timestamp

## ❓ Common Questions

**Q: Do I need to expand rows before printing?**  
A: No! Print view automatically shows all columns.

**Q: Can I print just one item?**  
A: Not directly. Use browser's print selection feature.

**Q: Why are some remarks highlighted yellow?**  
A: Yellow indicates budget deductions from other categories.

**Q: Can I export to Excel?**  
A: Not yet, but you can print to PDF and convert.

**Q: How many items can I view at once?**  
A: 20 items per page on screen, all items when printing.

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| Row won't expand | Refresh page, clear cache |
| Print shows collapsed view | Check print styles, try different browser |
| Arrow not rotating | Enable JavaScript |
| Details not showing | Check browser console for errors |
| Slow performance | Collapse all rows, reduce items per page |
